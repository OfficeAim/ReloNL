# Student Guides & Interactive Tools

ReloNL provides a suite of tools designed to navigate the complexities of the Dutch housing market. These tools are built to empower students with data and legal clarity.

## 📋 Key Student Resources

### 1. Pre-Arrival Solvency Dossier
A multi-step wizard that helps students prepare their documentation before landing.
- **Verification:** On completion, students receive a "✅ Verified Tenant Badge".
- **Trust:** A QR code is generated for landlords to scan and verify the student's solvency instantly.

### 2. Relocation Action Plan
An interactive checklist that guides students through every stage of their move:
- **Phase 1:** Pre-arrival (Visa, RNI, Phone setup).
- **Phase 2:** Arrival (BRP registration, Bank account).
- **Phase 3:** Month 1 (Insurance, local registrations).

### 3. Hospiteeravond Digitalizer
The "Hospiteeravond" (interview evening) is a unique Dutch tradition. ReloNL digitizes this experience:
- **Personality Profiling:** Sliders for sociability, cleanliness, and noise tolerance.
- **Matching:** Generates a "Roommate Profile" summary that students can copy-paste to landlords or student houses.

### 4. Smart Apartment Tracker
A data-driven tool to compare potential homes:
- **Visualization:** A scatter plot comparing **Price vs. Commute Time**.
- **Safety:** Filters for "Night Safety" zones in each of the 9 target cities.

## 🤖 Gemini AI Tools (Heuristic Fallback)
The platform includes AI-powered assistants for common tasks:
- **Scam Detector:** Analyzes listings for red flags (e.g., "no registration allowed").
- **Letter Generator:** Drafts landlord pitch letters in Dutch with English translations.
- **Legal Coach:** A chatbot interface for basic Dutch tenancy law queries.
